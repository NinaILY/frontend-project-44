### Project discription:
Brain Games - a set of console games

### Requirements
- Node.js >= 18
- npm

### Installation
npm install -g @hexlet/code

### Commands for launching games
- brain-even
- brain-calc
- brain-gcd
- brain-progression
- brain-prime
